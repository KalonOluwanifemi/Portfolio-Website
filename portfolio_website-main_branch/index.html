About

With over 4 years of experience, I have transitioned from frontend development to advanced cloud support and DevOps engineering. My journey began with the MERN stack (MongoDB, Express.js, React, Node.js), emphasizing robust application testing using Jest and React Testing Library. As I shifted to cloud support, I honed my expertise within the Microsoft Azure ecosystem, providing global customers with solutions through Azure API Management, App Service, Function Apps, Static Web Apps, Container Apps, and Azure Kubernetes Service (AKS).

Today, I specialize in cloud engineering and DevOps, focusing on CI/CD pipeline integration, container orchestration with Docker and Kubernetes, and automating deployments for scalable and reliable solutions. With a strong technical foundation in Azure tools, GitHub Actions, Azure DevOps, and modern web development technologies, I aim to deliver impactful solutions that optimize performance and efficiency for global teams.
